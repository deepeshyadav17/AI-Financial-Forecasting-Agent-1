# AI Financial Forecasting Agent

This project builds an AI agent to predict stock market trends using:

- ARIMA (baseline)
- LSTM (deep learning)
- Reinforcement Learning (trading agent)

## Features
- Multi-stock dataset (NIFTY 50)
- Data cleaning pipeline
- Forecasting models
- Trading simulation
- Dashboard (coming soon)

## Tech Stack
- Python
- TensorFlow
- FastAPI
- React
